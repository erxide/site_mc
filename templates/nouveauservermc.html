Comment rejoindre

Important, pour pouvoir rejoindre nos serveurs minecraft, il faut avoir minecraft java edition et openvpn intallé sur votre ordinateur.


Se connecter au vpn

Veuillez aussi, telecharger votre fichier de conf de openvpn ici, pour pouvoir rejoindre le vpn.

Windows

  • Une fois openvpn installé, et votre fichier de conf téléchagé, faite clic droit sur l'icône openvpn, puis aller dans Import, et clicker sur Import file...

    puis chercher le fichier que vous venez de téléchager, et importé le.

  • Ensuite clicker sur connecter, dans le menu qui viens de s'ajouter dans votre openvpn

  • Si le l'ecran de l'icone openvpn devient vert alors vous etes normalemnt connecter au vpn ^^.

Linux

  • ouvrir un terminal

  • $ sudo openvpn votre-fichier.ovpn

    or

    $ sudo openvpn --config votre-fichier.ovpn

Tester le VPN


Dans un terminal faite : $ ping 10.8.0.1

Si aucun des packets n'est perdu alors vous etes bien connecté au vpn.
